Python装饰器

  1. 把装饰器当作一种函数Plus模块来使用
  2. 装饰器的关键字符是 @
  3. 装饰器作用于函数，同时由函数构成，也属于函数式编程
  4. 装饰器定义时需要注意函数传参数的问题（*args,**kw）

None是什么鬼

None就是None

使用bool()操作一个自定义对象时，返回的是True还是False由什么决定？

当一个类的len返回为0，同时bool没有被重写时，bool()这个类就会获得False的返回值，
当一个类的len返回不为0，bool没有被重写，bool()返回值就为True，若bool被重写，则返回值为bool的返回值。
