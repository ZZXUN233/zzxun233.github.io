什么是Markdown

数学公式的插入

参考网站

Cmd Markdown 编辑阅读器支持 $\LaTeX$ 编辑显示支持，例如：$\sum_{i=1}^n a_i=0$，访问 MathJax 以参考更多使用方法。

$\LaTeX$ 的数学公式有两种：行中公式和独立公式。行中公式放在文中与其它文字混编，独立公式单独成行。

自动编号后的公式可在全文任意处使用 \eqref{eq:公式名} 语句引用。
